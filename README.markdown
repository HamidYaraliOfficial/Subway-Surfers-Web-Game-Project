# Subway Surfers Web Game Project

## English

# 🎮 Subway Surfers WebGL Game

A lightweight web-based launcher for a Unity WebGL version of Subway
Surfers.\
This project provides an HTML interface that loads Unity WebGL assets,
prepares split files, and runs the game directly in a modern browser.

## ✨ Features

-   Unity WebGL game loader
-   Automatic asset merging for split files
-   Fullscreen support
-   Browser-based gameplay
-   Simple HTML deployment
-   Optimized loading process

## 📋 Requirements

Before running the project, make sure you have:

-   A modern web browser:
    -   Google Chrome
    -   Microsoft Edge
    -   Mozilla Firefox
-   A local web server (recommended)

## 🛠 Installation

### 1. Install Python

Download and install Python from the official website.

### 2. Install required tools

Install the required Python package:

``` bash
pip install http-server
```

If you use Node.js, you can install a local server with:

``` bash
npm install -g http-server
```

## ▶️ Run the Project

Open the project folder and start the server:

``` bash
http-server
```

Then open the displayed local address in your browser.

Example:

``` text
http://localhost:8080
```

## 📁 Project Structure

    .
    ├── index.html
    ├── master-loader.js
    ├── 4399.z.js
    ├── Unity WebGL files
    └── Assets

## 🎯 Controls

-   Press **F** to toggle fullscreen mode.

# 🎮 Subway Surfers WebGL 游戏项目

这是一个基于 Unity WebGL 的 Subway Surfers 网页游戏项目。

项目通过 HTML 页面加载 Unity WebGL
资源，自动处理分割文件，并在现代浏览器中运行游戏。

## ✨ 功能特点

-   Unity WebGL 浏览器运行
-   自动合并分割资源文件
-   支持全屏模式
-   简单部署
-   快速加载

## 📋 环境要求

需要准备：

-   现代浏览器：
    -   Google Chrome
    -   Microsoft Edge
    -   Firefox
-   本地 Web 服务器

## 🛠 安装步骤

### 1. 安装 Python

请先安装 Python 环境。

### 2. 安装所需工具

使用以下命令安装：

``` bash
pip install http-server
```

或者使用 Node.js：

``` bash
npm install -g http-server
```

## ▶️ 启动项目

进入项目目录后运行：

``` bash
http-server
```

然后在浏览器打开服务器显示的地址。

示例：

``` text
http://localhost:8080
```
